## Usage

1. Launch KairoX GUI
2. Select "Phishing Attack"
3. Choose a template from dropdown
4. Enter custom domain and keyword
5. Click "Start Attack"
6. URLs generated automatically

## Template Structure

Each template contains:
- `index.html` - Main phishing page
- Inline CSS for styling
- JavaScript for form handling
- Auto-redirect to real site

## Customization

Templates can be customized by editing:
- Colors and fonts in `<style>` section
- Text and placeholders
- Redirect URLs in JavaScript
- Logo and branding elements

## Adding New Templates

To add a new template:

1. Create folder: `templates/newsite/`
2. Create `index.html` with form
3. Form action: `../login.php`
4. Form fields: `username` and `password`
5. Add redirect URL in JavaScript
